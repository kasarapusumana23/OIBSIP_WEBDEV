# 🔐 Lumora – Login Authentication System

---

## 🎯 Project Overview

Lumora is a simple Login Authentication System developed using HTML, CSS, and JavaScript. It allows users to register, log in, access a protected dashboard, and log out securely.

---

## ✨ Features

✅ User Registration

✅ User Login

✅ Authentication Validation

✅ Secure Dashboard Access

✅ Logout Functionality

✅ Local Storage Integration

✅ Responsive Design

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Local Storage API
