# 🌸 Daily Bloom – To-Do Web Application

---

## 🎯 Project Overview

Daily Bloom is a modern task management application that helps users organize daily activities efficiently through task creation, editing, completion tracking, and local storage support.

---

## ✨ Features

✅ Add Tasks

✅ Edit Tasks

✅ Delete Tasks

✅ Mark Tasks as Completed

✅ Separate Pending & Completed Tasks

✅ Local Storage Support

✅ Responsive Design

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Local Storage API
