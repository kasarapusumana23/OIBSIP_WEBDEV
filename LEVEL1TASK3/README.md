# 🌡️ Temperature Converter Website

---

## 🎯 Project Overview

A temperature converter web application that converts values between Celsius, Fahrenheit, and Kelvin using JavaScript.

---

## ✨ Features

✅ Celsius ↔ Fahrenheit Conversion

✅ Celsius ↔ Kelvin Conversion

✅ Fahrenheit ↔ Kelvin Conversion

✅ Instant Results

✅ Input Validation

✅ Responsive Design

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
