# 🍕 PizzaCraft - Pizza Delivery App

## 📌 Overview
PizzaCraft is a full-stack pizza delivery web application that allows users to browse pizzas, add items to cart, register/login, and place orders.

## 🚀 Features
- User Registration & Login
- JWT Authentication
- Browse Pizza Menu
- Add to Cart
- Order Management
- Responsive UI

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB

## 📂 Project Structure

PizzaCraft/
├── client/
└── server/

## ⚙️ Installation

### Clone Repository
```bash
git clone <repository-url>
```

### Frontend
```bash
cd client
npm install
npm run dev
```

### Backend
```bash
cd server
npm install
npm start
```

## 🔐 Environment Variables

Create a `.env` file inside the server folder and add:

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```
